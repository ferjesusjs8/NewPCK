# NewPCK
How to generate new Key

1.
git config --global user.name "username_xpto"

2.
git config --global user.email "email_xpto@provider_xpto.com"

3.
 ssh-keygen
 and then hit 3 'ENTER button'
 
 4.
 cat ~/.ssh/id_rsa.pub
copy the ssakey

5.
Put it into github.com > settings > new SSH KEY

6.
DONE!
